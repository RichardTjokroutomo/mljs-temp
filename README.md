This is a temporary repository that contains the compiled mljs library and the models it depends on. 

> Note that `index.js` is generated by the typescript compiler, so it is **not** easy to read.

---

At present, the library works on the following condition:

```html
<div class="mljs-container">
    <img src="...">
</div>
```

The image element is wrapped by a `<div>` element with `mljs-container` class.

Then, include `mljs-3d.js` in the `html`:

```html
<script type="module" src="/path/to/mljs/lib/mljs-3d.js"></script>
```
